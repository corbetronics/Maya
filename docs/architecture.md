# Project MAYA Architecture

Project MAYA is organized as a clean skeleton for a realtime AI podcast guest.

- `backend/` contains the FastAPI application, typed settings, database infrastructure, and service configuration objects.
- `frontend/` contains the React, TypeScript, and Vite studio interface.
- `brain/` contains domain contracts for future conversation intelligence.
- `tests/` contains smoke tests that verify the application is importable and routable.

Business behavior is intentionally absent from this initial build.
